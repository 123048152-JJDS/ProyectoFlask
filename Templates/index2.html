¿Cómo funciona?

  • Frontend (JavaScript): Interfaz de usuario que permite agregar, completar y eliminar tareas
  • Backend (Python Flask): Maneja la lógica de negocio y almacenamiento de datos
  • Comunicación: El navegador envía solicitudes HTTP al servidor Python
  • Persistencia: Las tareas se almacenan en el servidor para mantener el estado

Implementación del Servidor Python

# app.py - Backend en Python con Flask
from flask import Flask, jsonify, request, render_template

app = Flask(__name__)

# Almacenamiento de tareas (en producción usar base de datos)
tareas = {
    "pendientes": [
        {"id": 1, "texto": "Estudiar para el examen de matemáticas"},
        {"id": 2, "texto": "Comprar víveres en el supermercado"}
    ],
    "completadas": [
        {"id": 3, "texto": "Hacer ejercicio en el gimnasio"}
    ]
}

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/api/tareas', methods=['GET'])
def obtener_tareas():
    return jsonify(tareas)

@app.route('/api/tareas/pendientes', methods=['POST'])
def agregar_tarea():
    nueva_tarea = request.json
    if not nueva_tarea or 'texto' not in nueva_tarea:
        return jsonify({"error": "Datos inválidos"}), 400
    
    # Crear nueva tarea con ID único
    nueva_id = max([t['id'] for t in tareas['pendientes'] + tareas['completadas']], default=0) + 1
    tarea = {"id": nueva_id, "texto": nueva_tarea['texto']}
    
    tareas['pendientes'].append(tarea)
    return jsonify(tarea), 201

@app.route('/api/tareas/completar/', methods=['PUT'])
def completar_tarea(tarea_id):
    # Buscar tarea en pendientes
    for i, tarea in enumerate(tareas['pendientes']):
        if tarea['id'] == tarea_id:
            # Mover a completadas
            tarea_completada = tareas['pendientes'].pop(i)
            tareas['completadas'].append(tarea_completada)
            return jsonify(tarea_completada), 200
    
    return jsonify({"error": "Tarea no encontrada"}), 404

@app.route('/api/tareas/', methods=['DELETE'])
def eliminar_tarea(tarea_id):
    # Buscar en pendientes
    for i, tarea in enumerate(tareas['pendientes']):
        if tarea['id'] == tarea_id:
            tareas['pendientes'].pop(i)
            return jsonify({"mensaje": "Tarea eliminada"}), 200
    
    # Buscar en completadas
    for i, tarea in enumerate(tareas['completadas']):
        if tarea['id'] == tarea_id:
            tareas['completadas'].pop(i)
            return jsonify({"mensaje": "Tarea eliminada"}), 200
    
    return jsonify({"error": "Tarea no encontrada"}), 404

if __name__ == '__main__':
    app.run(debug=True)

Endpoints de la API

  • GET /api/tareas - Obtener todas las tareas
  • POST /api/tareas/pendientes - Agregar nueva tarea pendiente
  • PUT /api/tareas/completar/{id} - Mover tarea a completadas
  • DELETE /api/tareas/{id} - Eliminar tarea
